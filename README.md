 📌 Overview

This project focuses on ensuring data accuracy, consistency, and reliability by performing comprehensive data quality checks and validation using Microsoft Excel and Python. It demonstrates how raw datasets can be cleaned, validated, and prepared for downstream analytics and decision-making.

🎯 Objectives
Identify and fix data quality issues (missing, duplicate, inconsistent data)
Perform data validation checks to ensure integrity
Automate validation processes using Python
Improve dataset reliability for analytics and reporting

📂 Dataset

Type: Structured business dataset (sales / operations / customer data)
Size: Large dataset (multiple columns & records)
Common issues handled:
Missing values
Duplicate entries
Incorrect formats
Outliers and anomalies

⚙️ Project Workflow

1️⃣ Data Cleaning (Excel)
Removed duplicates using Excel tools
Handled missing values using filtering and conditional logic
Standardized formats (dates, numbers, categories)
Used Data Validation rules to restrict incorrect inputs

2️⃣ Data Validation (Excel)

Applied data validation checks (drop-downs, constraints)
Used conditional formatting to highlight anomalies
Created summary sheets to track data errors

3️⃣ Automated Validation (Python)

Used Pandas & NumPy for data processing
Identified missing values, duplicates, and inconsistencies
Built reusable scripts for validation checks

Generated automated reports for data quality metrics
📊 Key Checks Performed
✅ Missing Value Analysis
✅ Duplicate Detection
✅ Data Type Validation
✅ Range & Constraint Checks
✅ Outlier Detection
✅ Consistency Checks across columns

🛠️ Tech Stack

Microsoft Excel – Data cleaning & validation
Python – Automation & advanced analysis
Pandas – Data manipulation
NumPy – Numerical operations
